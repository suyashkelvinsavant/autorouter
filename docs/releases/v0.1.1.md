# AutoRouter v0.1.1

Patch release that restores a clean strict-Clippy CI baseline after the initial public release.

## Windows installation

Download `AutoRouter_0.1.1_x64-setup.exe`, verify its SHA-256 checksum, then run the installer.

```text
48C9496E7AD6B828D1FD21005185FAF33CFE76193BA7E4BDAFFE10A9B7A69322  AutoRouter_0.1.1_x64-setup.exe
```

This installer is not code-signed. Verify the checksum before proceeding; Windows SmartScreen may show a warning.

## Changes

- Fixed all strict-Clippy findings across the workspace, including streaming, session storage, TUI, dashboard, and test code.
- Kept the public gateway behavior unchanged while simplifying internal interfaces.
