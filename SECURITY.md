# Security Policy

## Supported versions

Security fixes are applied to the latest released version of AutoRouter.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability, leaked credential, or bypass of gateway authentication.

Instead, use GitHub’s private security advisory reporting for this repository. Include a clear reproduction, affected version, impact, and any suggested mitigation. You should receive an acknowledgement within seven days.

If private reporting is unavailable, contact the repository maintainers through a private GitHub channel and avoid including secrets in the report.

## Scope

Reports are especially welcome for issues involving bearer authentication, local network exposure, secret handling, configuration persistence, installer integrity, and request forwarding.
